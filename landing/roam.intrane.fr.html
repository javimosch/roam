How it works

01 · dispatch

Send it and walk away

roam send --to prod-01 --goal "…" ships the binary to the box and runs the agent detached. One static binary — no Python, no runtime on the server.

02 · it parks

Nothing destructive runs unasked

Hit rm -rf, git push --force, DROP TABLE, shutdown… and the agent freezes and asks. Budgets and a workdir sandbox bound the rest.

03 · you decide

From anywhere, in one tap

Approve, deny (the agent adapts), or stop. See every agent across every box — live status, token cost, the full journal — in one place.

Why not just ssh?

Because you're not always at your laptop.

The free CLI already lets you approve over ssh. But the agent parks at 2am, or you're on café wifi that can't reach the box. The hosted layer is the agent phoning home — so the decision comes to you, wherever you are.
